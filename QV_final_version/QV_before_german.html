Sie werden jetzt über diejenigen politischen Vorschläge abstimmen, die Sie gerade unterstützt oder abgelehnt haben. Sie können nun im Rahmen des quadratischen Abstimmungsmechanismus mehrere Stimmen für denselben Politikvorschlag benutzen.

Wichtig: Diese Stimmen kaufen Sie mit "vote-credits" (§). Die Kosten in "vote-credits" für die Abstimmung zu einem Thema sind das Quadrat der Anzahl der Stimmen, die Sie zu diesem Thema abgeben möchten.

Beispiel: Wenn Sie dreimal für/gegen den Mindestlohn stimmen möchten, geben Sie 9 credits für dieses Thema aus.

Versuchen Sie, die Credits für Themen auszugeben, die Ihnen wichtig sind! Sie können sich auch dafür entscheiden, nichts für ein Thema auszugeben.
