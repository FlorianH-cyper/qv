Willkommen zu diesem Experiment!

Sie werden heute zum Vergleich zweier Wahlmechanismen beitragen, indem Sie über einige Politikvorschläge abstimmen:

1. Majority Voting:

Jeder darf pro Wahl eine Stimme abgeben, die Mehrheit an Stimmen entscheidet.

2. Quadratic Voting:

Es ist möglich, mehrere Stimmen für eine Wahl/ein Thema abzugeben. Erhält ein Vorschlag mehr Stimmen dafür als dagegen, wird er umgesetzt, unabhängig von der Anzahl der Personen, die tatsächlich dafür oder dagegen gestimmt haben.
